# Gera_MovementShooter

## Description
This is a project I was working on the for end of my CG Spectrum Game Programming Course.
 
## Controls
* WASD - Move
* Q - Throw Grenade
* E - Use Ability Pickup
* F - Interact / Pickup
* LeftClick - Shoot
* RightClick - Aim
* 1, 2, 3 - Equip Weapon
* ScrollWheel - Next/Previous Weapon
* LShift - Dash
* Space - Jump, Double Jump
